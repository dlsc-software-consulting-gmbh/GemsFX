### CalendarPicker

A control for selecting a date that replaces the default `DatePicker` that
ships with **JavaFX**. The month view that the user can bring up by clicking on the button on the right-hand
side of this control is an instance of `CalendarView`, which is more powerful than the normal view
used by `DatePicker`. The `CalendarView` allows the user to directly jump to a specific month or year.
